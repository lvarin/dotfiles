# My dotfiles

Configure linux workstation using Ansible.

## Base tools
 - wget
 - curl

### System

- git
- tmux
- vim
- zsh

## Bootstrap

Firt setup installation run the dot-bootstrap command.

```
$ ./bin/dot-bootstrap
```

After that you can run any scripts defined in the `$DOTFILES_PATH/bin`

```
$ dot-bootstrap
```
